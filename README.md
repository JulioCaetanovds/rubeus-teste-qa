# Teste Prático QA - RBS

Este repositório contém a resolução do teste prático para avaliação de qualidade de software, englobando testes exploratórios manuais e automação de testes.

## 📂 Estrutura do Projeto

A avaliação foi dividida em duas frentes principais para melhor organização:

1. **[Testes Exploratórios](./testes_exploratorios/)**: Relatórios detalhados de bugs, falhas de usabilidade e oportunidades de melhoria (UI/UX) encontrados nas páginas de avaliação.
   - [Avaliação da Página de Certificação](./testes_exploratorios/certificacao.md)
   - [Avaliação da Página do Site](./testes_exploratorios/site.md) (Em breve)

2. **[Testes Automatizados](./testes_automatizados/)**: Automação dos principais fluxos críticos identificados durante a exploração.

## 🛠️ Metodologia de Reporte
Os apontamentos manuais seguem o padrão exigido no escopo do teste, categorizando cada item por:
- **Tipo:** Correção, Nova funcionalidade ou Melhoria
- **Classificação:** Utilidade, Usabilidade ou Desejabilidade
- **Prioridade:** Alta, Média ou Baixa
